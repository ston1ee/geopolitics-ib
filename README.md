# 🌍 GeoPolIB — Interactive World Geopolitics

**From complete beginner to IB Politics expert** — an interactive web platform covering the full geopolitical map of the world, aligned with the IB Politics curriculum.

🔗 **Live Site:** https://ston1ee.github.io/geopolitics-ib/

---

## 📁 Repository Structure

```
geopolitics-ib/
├── index.html                    ← Main landing page
├── css/
│   └── style.css                 ← Global styles
├── js/
│   ├── main.js                   ← Progress tracking, flashcards, quiz engine
│   └── charts.js                 ← Chart.js data visualizations
├── modules/
│   ├── m1-foundations.html       ← Module 1: What is Politics? (Beginner)
│   ├── m2-state.html             ← Module 2: State & Sovereignty (Beginner)
│   ├── m3-power.html             ← Module 3: Power & Ideology (Intermediate)
│   ├── m4-conflicts.html         ← Module 4: Global Conflicts 2024-26 (Intermediate)
│   ├── m5-globalactors.html      ← Module 5: USA, China, Russia, EU, India (Advanced)
│   ├── m6-humanrights.html       ← Module 6: Human Rights & International Law (Advanced)
│   ├── m7-development.html       ← Module 7: Development & Economics (Advanced)
│   └── m8-expert.html            ← Module 8: Expert Geopolitical Theory (Expert)
└── games/
    ├── quiz.html                 ← 100+ question quiz (all modules)
    ├── map-game.html             ← Interactive country locator map
    ├── flashcards.html           ← 200+ term flashcard deck with spaced repetition
    └── timeline.html             ← Historical timeline 1648–2026
```

---

## 🚀 How to Enable GitHub Pages

1. Go to **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main**, folder: **/ (root)**
4. Click **Save**
5. Your site will be live at: `https://ston1ee.github.io/geopolitics-ib/`

## 🎮 Interactive Features

- **📊 Charts** — GDP comparisons, democracy indices, military spending (Chart.js)
- **🃏 Flashcards** — Click-to-flip concept cards in Danish & English
- **🧠 Quiz** — Multiple choice with instant feedback & score tracking
- **🗺️ Map Game** — Click countries on SVG world map
- **📅 Timeline** — Drag-and-drop events into chronological order
- **📈 Progress Bar** — localStorage-based module completion tracking

## 🛠️ Tech Stack

- Pure **HTML5, CSS3, JavaScript** (no build tools needed)
- **Chart.js** (CDN) — for data visualizations
- **LocalStorage** — for progress persistence
- **SVG** — for interactive map
- Works offline after first load
